Comandos basicos adiquiridos


  • procurar o diretório utilizando cd .. para retornar e cd nome/ para avançar
  • Depois pode-se criar um repositório direto no Github e clonar no seu local, por meio do git Clone "link gerado pelo Github"
  • adicionar manualmente os arquivo dentro da pasta
  • criar a pasta de controle do git com Git init
  • 1º analisar os arquivos dando um git status
  • 2º dar um git add * para adicionar os itens modificados e os itens estão prontos para ir ao controle de versão
  • 3º dar um git commit -m “comentario” Depois disso ira entender que é pra ir pro local e o comentario para fornecer informações, mas ainda não estará no servidor,
  • 4º dar um push origin main Agora sim ira subir pro servidor remoto
